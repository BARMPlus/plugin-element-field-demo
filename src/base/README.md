
封装element-ui 后台管理系统组件
===

### 简述

###### 1.此组件为方便个人开发而制作
###### 2.与element-ui结合
###### 3.目前提供组件：input,radio,checkbox,select,date-picker




### npm安装

```
npm install element-ui  --save
npm install element-field --save
```


### 引入
```
import ElementField from 'element-field';
Vue.use(ElementField);
```
